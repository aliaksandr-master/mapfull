# mapfull
Powerfull nodeJs library

## Getting started 

To install `mapfull` from npm, run:
```bash
npm install -g mapfull --save
```

Finally, use the library:
```js
mapfull = require('mapfull');

// do something helpful
```

### Enjoy!
